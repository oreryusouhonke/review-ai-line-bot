# 利用回数・キャンペーン設計ポリシー

レビュー職人の利用回数、ランク、達成タグは、Googleへの投稿完了数ではなくレビュー職人で口コミ文を作成した回数として扱います。

## 禁止すること

- Google口コミを書いたこと自体と引き換えに景品、抽選、報酬を提供すること。
- 実体験のない口コミ作成を促すこと。
- 虚偽レビューを増やす表現を使うこと。
- 星評価を誘導すること。

## 許容する設計

- レビュー職人の利用回数や活動実績に応じて、サービス内ランクや達成タグを付けること。
- 活動実績に応じて、スポンサー広告、イベント案内、先行機能案内を出すこと。
- 口コミ投稿の有無ではなく、本人の実体験をもとにした口コミ文作成支援の利用状況として集計すること。
